新建并配置容器

配置容器

  • -p 1000:8080

    • 8080是gitblit的http端口

  • -p 1001:8443

    • 8443是gitblit的https端口

  • -p 29418:29418

    • 29418是gitblit的ssh端口

  • -v /var/gitblit:/opt/gitblit-data

    • /opt/gitblit-data是gitblit的仓库目录
